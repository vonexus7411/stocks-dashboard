# Stock Analysis Application

A desktop application for analyzing stock market data.

## Setup

```bash
# Create virtual environment
make install

# Run tests
make test

# Start application
make run
```

## Development

```bash
# Format code
make format

# Run linters
make lint

# Run tests with coverage
make coverage
```

## Project Structure

```
stocks/
├── src/             # Source code
├── tests/           # Test files
├── data/            # Data storage
└── logs/            # Application logs
```
